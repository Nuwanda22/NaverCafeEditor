§ v0.1.1 §

[학번] 기능이 추가되었습니다.
+ Naming Rule에 [학번]을 설정하여 검색할 경우, 5자리로 일치되는 숫자 (ex : 10318)를 검색해줍니다.
검색된 학번은 다운로드 경로 설정에서 [학년], [반], [번호] 로 나누어서 설정할 수도 있습니다.

+ 로그인 시 tab을 이용하여 간단하게 입력창 변화가 용이하게 변경되었습니다.

+ 추가항목 (> 버튼)에 도움말 버튼이 추가되었습니다.
도움말 화면에서 사용법과 주의 및 공지사항을 확인 할 수 있습니다.

[ Manual ]

1. Login 버튼을 클릭하여 네이버 계정으로 로그인 합니다.

2. 검색을 시행할 타겟 카페 주소를 입력합니다.

3. 검색할 문서들의 게시 허용 기간을 설정합니다.

4. 검색될 문서들의 제목 규칙 (Naming Rule)을 설정해줍니다.
 
5. Search 버튼을 클릭하여 게시글을 검색합니다.

6. 현재 설정되어있는 Naming Rule을 저장 할  수 있습니다.
    (저장된 템플릿은 Open Template 버튼을 클릭하여 열 수 있습니다.)

7. 검색이 완료 된 후에     버튼을 클릭하여 설정환경을 변경합니다.

8. 파일이 저장될 이름 형식을 지정합니다.  (Naming Rule에 등록된 텍스트 + 사용자 입력 텍스트)

9. 파일이 저장될 로컬 위치를 설정 합니다.

10. Download 버튼을 클릭하여 파일을 다운로드 합니다.



[ Notice ]

Naming Rule에서 변수와 변수는 함께 붙어서 사용할 수 없습니다.

가장 첫 칸에는 변수가 들어올 수 없습니다.

아직은 제출기한 설정을 Day - Day 만 가능하고, 시간별 설정은 불가능합니다.
	-> (v0.1.1) 이제 시간별 설정도 이용할 수 있습니다.
